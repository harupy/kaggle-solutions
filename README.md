# Kaggle Solutions

https://harupy.github.io/kaggle-solutions/

## Setup

```bash
# This step is optional.
pip install pipenv

# Install dependencies.
pipenv install

# Activate the environment.
pipenv shell
```

## How to fetch solution data

```bash
python tools/fetch_discussion.py -u <discussion_url> -t <title>
```

## UI

```bash
cd client
npm install
npm start
```
